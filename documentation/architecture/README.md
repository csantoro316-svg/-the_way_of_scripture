# Repository Architecture

Use this folder for stable site structure, content workflow, and implementation rules that should persist beyond one session.

## Current Documents

- `CONTENT_EXECUTION_STANDARD.md` - canonical workflow for turning study content into study pages and linked passage pages, including the default Bible version standard.
- `COMMENTS_AND_FEEDBACK_STANDARD.md` - canonical policy and future system shape for public comments, feedback intake, moderation, and revision posture.
- `WEBSITE_STYLE_STANDARD.md` - canonical visual, layout, and readability standard for the site.
