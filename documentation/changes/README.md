# Change Docs

This folder is the running change log for repo-scoped work.

## Naming

Use:

`YYYY-MM-DD-short-description.md`

## Usage

- Add a change doc when implemented behavior or repo workflow changed.
- Keep each file focused on one meaningful slice.
- Use these docs as the continuity source for the next session.

## Template

Start from:

- `documentation/changes/templates/template.md`
